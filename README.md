## 文件功能描述

* data_loader：用于数据导入以及预处理
* mains：用于存放训练、测试、预测代码文件的入口
* model：存放神经网络模型或者虚拟网络嵌入框架
* util：用于存放一些功能文件，比如配置文件，日志文件，功能函数文件
* main.py：作为程序启动入口
